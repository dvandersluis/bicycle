# Bicycle

🚲🃏

Ruby pure-OO implementation of a deck of cards.

## Usage

```ruby
deck = Bicycle::Deck.new.shuffle
hands = deck.deal(4, 5) # 4 hands of 5 cards
```

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bicycle'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cards
